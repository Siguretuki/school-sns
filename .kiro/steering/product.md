# Product Overview

学校向けSNSアプリケーション。学生・教師間のコミュニケーションとコンテンツ共有を支援する。

## Core Capabilities

- **ユーザー管理**: ロールベース（ADMIN/STUDENT/TEACHER）の認証・認可
- **コンテンツ投稿**: Artifacts（記事/作品）とScraps（短い投稿）の作成・共有
- **ソーシャル機能**: フォロー関係、コメント、メンション
- **タグ・検索**: コンテンツのタグ付けと検索機能
- **ワールド**: グループ/コミュニティ機能

## Target Use Cases

- 学生が作品や学習成果物を投稿・共有
- 教師と学生間のコミュニケーション
- 学内コミュニティでの情報交換
- 学習コンテンツの発見と検索

## Value Proposition

- 教育機関に特化したセキュアなSNS環境
- ロールベースのアクセス制御
- シンプルで使いやすいインターフェース
