# カードデザイン改修報告

## 概要
ScrapsおよびArtifactsコンテンツの表示を、視認性を高めてモダンなUIにするため、既存のカードデザインを大幅に刷新しました。

## 1. Scraps カードデザイン

### 変更前
- **レイアウト**: 垂直方向の単純な積み上げ。
- **情報量**: ユーザー名、アバター、本文のみ。
- **スタイル**: 背景色が薄いグレーで、コンテンツ間の区切りが弱い。
- **機能**: いいねやコメント数の表示がない。

### 変更後
- **レイアウト**: 
  - 白背景、角丸、淡い影付きのカード型コンテナを採用し、各投稿を明確に区別。
  - 白背景、角丸、淡い影付きのカード型コンテナを採用し、各投稿を明確に区別。
  - アバター（絶対配置）と本文リンクを分離し、クリック領域を最適化。
  - ヘッダーおよびフッターの実装に `IconWithLabel` コンポーネントを使用し、既存のコード構造を維持。
- **情報追加**:
  - **日時**: 「XX時間前」という固定のプレースホルダーを表示（実装方針保留中）。
  - **アクション**: 右上に「三点リーダー（メニュー）」アイコンを配置（プレースホルダー）。
- **フッター**:
  - **エンゲージメント**: ハート（いいね数）と吹き出し（コメント数）のアイコンを追加。
  - **スタイル**: アイコンにホバー時の色変化（ピンク/青）を追加。

## 2. Artifacts カードデザイン

### 変更前
- **レイアウト**: テキスト主体のリスト形式。
- **情報量**: タイトル、日付、作成者のみ。
- **スタイル**: シンプルな枠線のみで、視覚的なインパクトが弱い。

### 変更後
- **レイアウト**: 
  - **左側（サムネイル）**: 画像用のプレースホルダーエリアを確保（現在はアイコン表示）。
  - **右側（コンテンツ）**: 詳細情報を構造化して配置。
  - フレックスボックスを使用した横並びレイアウトに変更。
- **情報追加**:
  - **バッジ**: 「ARTIFACT」という緑色のバッジを追加し、コンテンツの種類を明示。
  - **要約**: AIによる要約文（`summaryByAI`）を表示（2行で省略）。
  - **日時**: 「XXh XXm」という固定のプレースホルダーを表示（実装方針保留中）。
  - ※タグ表示およびいいね数は未実装のため削除済み。
- **フッター**:
  - 作成者情報を下部に配置し、メインコンテンツと分離。

## 3. 技術的変更点

### 日付表示について
- パッケージマネージャの不整合エラーのため `date-fns` の導入を見送りました。
- カスタムユーティリティによる実装も一時保留とし、現在はデザイン確認用に「XXh XXm」という静的な文字列を表示しています。
- 実装方針についてリーダーの判断待ちの状態です。
